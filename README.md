# 🐞 Bug Tracker System (Python)

A simple command-line based Bug Tracker built using Python.  
This application allows users to add, view, update, and delete bugs with priority levels and timestamps.

---

## 🚀 Features

- Add new bugs with title and priority  
- View all bugs with status and timestamp  
- Update bug status (Open → Closed)  
- Delete bugs  
- Persistent storage using JSON (data saved even after closing the program)

---

## 🛠️ Technologies Used

- Python  
- JSON (for data storage)  
- datetime module  

---

## ▶️ How to Run

1. Clone the repository
2. Navigate to the folder
3. Run the program
